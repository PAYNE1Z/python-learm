## 第二章 常用模块 练习


### 需求
    需求如下：
    目录结构为
     .
     ├── account
     │   └── luffy.json
     ├── bin
     │   └── start.py
     └── core
     |   └── withdraw.py
     └── logs
         └── bank.log
    当执行start.py时，出现交互窗口
    
      ------- Luffy Bank ---------
      1.  账户信息
      2.  转账
      3.  提现
      
    选择1 账户信息 显示luffy的当前账户余额。
    选择2 转账 直接扣掉75万和利息费用并且tesla账户增加75万
    选择3 提现 提现金额应小于等于信用额度，利息为5%，提现金额为用户自定义。
